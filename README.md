# AndCommon



[![](https://jitpack.io/v/h4h4da2/andCommon.svg)](https://jitpack.io/#h4h4da2/andCommon)

1. #### 关于本项目

   - 对以前项目内的组件进行整理，进行二次封装，方便之后快速立项迭代。
   - 对于一些新框架尝鲜和探索。

2. #### 项目内的组件

   1. 列表组件
   2. 状态控制组件
   3. 状态栏组件
   4. 全局状态管理组件（待开发）
   5. kotlin 相关的拓展
   6. 待添加...
   
3. #### 引用


   ```
   allprojects {
   	repositories {
   		...
   		maven { url 'https://jitpack.io' }
   	}
   }
   
   dependencies {
   
       implementation 'com.github.h4h4da2:andCommon:version'
   }
   ```
